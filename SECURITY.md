# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.0     | :white_check_mark: |
| 1.6.x   | :white_check_mark: |
| 1.x     | :x:                |

## Reporting a Vulnerability

If you find a bug which is safety related, you should post a new issue named **Security Issue**, and I will check it as soon as possible.
